# FOOL Project

A compiler implementation for the FOOL (Functional Object-Oriented Language) programming
language.

## Overview

FOOL is an educational programming language that combines functional and object-oriented
programming paradigms. This project implements a complete compiler for FOOL, including
lexical analysis, parsing, semantic analysis, and code generation.

## Features

- **Lexical Analysis**: Tokenization of FOOL source code
- **Syntax Analysis**: Context-free grammar parsing
- **Semantic Analysis**: Type checking and symbol table management
- **Code Generation**: Translation to intermediate or target code
- **Error Handling**: Comprehensive error detection and reporting
